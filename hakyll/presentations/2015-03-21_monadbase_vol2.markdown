# 今回のアジェンダ

* モナド変換子とは $T : \left| \mathrm{Mon}(\mathrm{Hask}^{\mathrm{Hask}}) \right| \to \mathrm{Mon}(\mathrm{Hask}^{\mathrm{Hask}})$ なる関手と、埋め込み関手 $\mathrm{In}$ について $\mathrm{lift}^T : \mathrm{In} \Rightarrow T$ なる自然変換
* プログラムにおいては各モナドにおける操作の持ち上げが重要な関心ごと
